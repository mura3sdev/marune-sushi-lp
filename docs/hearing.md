## 多言語機能の仕様
- ヘッダーの JA / EN トグルで全文を日英切替（Vanilla JS・辞書オブジェクト方式）
- 選択言語は localStorage に保存し、再訪時に復元
- html の lang 属性も切替（アクセシビリティ・SEO配慮）
- 翻訳辞書は js/script.js の I18N オブジェクトに集約（クライアントが文言修正しやすい設計）
- 本番でURL分割（/en/）にする場合は hreflang 対応へ拡張可能とコメントに記載
